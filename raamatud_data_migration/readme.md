Excelist võetud TSV formaadingus faili lugemine SQL adnmebaasi INSERT käskudeks.

Kasutus:
1) projektil on kaasas TSV formaadingus fail, raamatud_nimekiri_tab.txt
2) Antud faili formaadingu alusel olev skript loob INSERT käsud iga TSV faili rea suhtes (v.a esimene rida)
3) Skripti käivitamise tulemusel ilmub uus fail nimega "sql_import.sql".
4) Skripti tulemusel saadud sql on sisestatav andmebaasi.
